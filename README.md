# tertiusdolens

Academic concept page for *tertius dolens*, the suffering third, based on the 2026 *Organization Science* paper by Adam Tatarynowicz and Thomas Keil titled, “*Tertius Dolens*: Interalter Conflict and Its Negative Impact on Broker Performance.”

## About

This repository hosts the source files for http://tertiusdolens.com, a minimal academic website dedicated to the new sociological concept of *tertius dolens*, or the suffering third.

*Tertius dolens* describes a brokering position in which conflict between alters turns an apparently favorable network position into a significant liability. The concept is developed in:

Tatarynowicz, Adam, and Thomas Keil. 2026. “*Tertius Dolens*: Interalter Conflict and Its Negative Impact on Broker Performance.” *Organization Science*. https://doi.org/10.1287/orsc.2024.19168

## Purpose

The website serves as a simple scholarly resource that provides:

- a concise explanation of the concept
- bibliographic information and links to the paper
- related materials, talks, and future work
- a stable public reference point for the idea

## Repository contents

- `index.html` – main homepage
- `styles.css` – site styling
- `assets/` – images and other static files
